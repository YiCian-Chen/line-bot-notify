## git使用介紹
#### 將遠端資料下載到本機
git pull --rebase

#### 上傳到遠端步驟
git add .

git commit -m "註解"

git push

#### PIP 安裝 requirements.txt 的套件
在此檔案目錄下輸入以下指令，即可安裝套件

pip install -r requirements.txt

之後檔案所需要使用的套件要新增進去
